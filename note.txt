1. IDE BISNIS STARTUP
================================================================

APA ITU SIGNALCORE?
-------------------
SignalCore adalah platform Video Conferencing berbasis self-hosted
yang memungkinkan perusahaan, institusi, dan pemerintah menjalankan
infrastruktur komunikasi video mereka sendiri — tanpa ketergantungan
pada Zoom, Google Meet, atau Microsoft Teams.

MASALAH YANG DISELESAIKAN:
---------------------------
- Zoom Enterprise = $250/user/tahun (100 user = Rp 390 juta/tahun)
- Data rapat keluar ke server luar negeri (melanggar UU PDP 2022)
- Tidak bisa dikustomisasi untuk kebutuhan internal perusahaan
- Performa drop saat user > 50 (arsitektur mesh biasa tidak mampu)
- Tidak ada solusi lokal sekelas Zoom di Indonesia

SOLUSI SIGNALCORE:
------------------
- Self-Hosted: data tidak pernah keluar dari server klien
- SFU Mediasoup: support 100+ user, latensi < 300ms
- JWT RS256: keamanan enterprise-grade
- Redis load balancing: pilih node paling ringan otomatis
- Whiteboard kolaboratif real-time (sudah ada di codebase)
- Screen sharing (sudah ada di codebase)
- Grid adaptif sesuai jumlah peserta aktif

TARGET PASAR:
-------------
1. Institusi Pendidikan (4.500+ kampus swasta — ENTRY POINT)
   - Rata-rata bayar Zoom Rp 15–50 juta/tahun
   - Keputusan lebih cepat, tidak perlu tender panjang
   - Pitch: "Hemat 70%, data di server kampus sendiri"

2. Klinik & Telemedicine
   - PERMENKES No.20/2019 wajibkan enkripsi data pasien
   - Pitch: "Konsultasi dokter-pasien yang benar-benar privat"

3. Pemerintah Daerah & BUMN
   - Inpres digitalisasi 2024 + anggaran Rp 27,3 triliun
   - UU PDPRI 2022 dorong kedaulatan data
   - Pitch: "Rapat dinas berdaulat, sesuai regulasi nasional"

4. Developer / API (jangka panjang)
   - Jual SignalCore sebagai Video API terintegrasi
   - Model: pay-per-minute seperti Twilio tapi lebih murah

--------------
